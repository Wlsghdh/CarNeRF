## 역할 배정

### 1. 3D 파이프라인 담당

**핵심 업무**: 영상 → 3D Gaussian Splatting 모델 생성 전 과정

| 구분 | 내용 |
|------|------|
| **담당 범위** | 프레임 추출, COLMAP SfM, 배경 제거, 깊이맵, 3DGS 학습, 모델 export |
| **주요 파일** | `scripts/run_pipeline.py`, `run_colmap.py`, `train_gaussian.py`, `export_model.py`, `extract_frames.py`, `remove_background.py`, `generate_depths.py`, `train_hq.py` |
| **의존 기술** | PyTorch, CUDA, pycolmap, 3D Gaussian Splatting, rembg, OpenCV |
| **현재 상태** | 파이프라인 동작 확인 (PSNR 31.88), HQ모드 구현 완료 |

**할 일 목록**:
- [ ] 3D 품질 향상 (목표 PSNR 34+)
- [ ] floater(부유물) 제거 알고리즘 개선
- [ ] 차량 내부 촬영 → 내부 3D 모델 생성 파이프라인
- [ ] 외부+내부 모델 통합 뷰어 구현
- [ ] 처리 시간 최적화 (현재 표준 45분, HQ 140분)
- [ ] 다양한 차종/환경에서의 안정성 테스트
- [ ] 모델 경량화 (모바일 대응)

### 2. AI 담당 (결함탐지 + 가격예측)

**핵심 업무**: 차량 결함 자동 탐지 + AI 기반 시세 예측 모델 개발

| 구분 | 내용 |
|------|------|
| **담당 범위** | YOLO 결함탐지 모델, 가격예측 ML 모델, 학습 데이터 수집/전처리, AI 진단서 생성 |
| **주요 파일** | `scripts/train_defect_model.py`, `scripts/train_price_model.py`, `scripts/crawling/encar_crawler.py`, `backend/app/api/defect.py`, `backend/app/api/predict.py` |
| **의존 기술** | YOLOv8, SAM, LightGBM, scikit-learn, pandas |
| **현재 상태** | 결함탐지: 데모 데이터만 존재 / 가격예측: 휴리스틱 기반 (ML 모델 미구현) |

**할 일 목록**:
- [ ] 차량 결함 학습 데이터셋 수집 및 라벨링 (scratch, dent, paint_damage, corrosion 등)
- [ ] YOLOv8 결함탐지 모델 학습 및 검증
- [ ] 3D 렌더링 → 멀티뷰 2D → 결함 탐지 파이프라인 구현
- [ ] 결함 위치를 3D 좌표로 매핑 (뷰어에서 마커 표시)
- [ ] 엔카/KB차차차 데이터 크롤링으로 가격 데이터셋 구축
- [ ] LightGBM 가격예측 모델 학습 (brand, model, year, mileage, 결함점수 등)
- [ ] 결함 심각도별 가격 감가 반영 모델

## 역할 간 협업 포인트

```
┌──────────────┐     모델 파일      ┌──────────────┐
│  1. 3D 파이프 │ ──────────────▶   │  3. 웹 플랫폼 │
│    라인 담당   │                   │     담당       │
└──────┬───────┘                    └──────▲───────┘
       │                                   │
       │  3D 렌더 → 멀티뷰                  │ API 연동
       │                                   │
┌──────▼───────┐     진단 결과       ┌──────┴───────┐
│  2. AI 담당   │ ──────────────▶   │  3. 웹 플랫폼 │
│ (결함+가격)    │                   │     담당       │
└──────────────┘                    └──────────────┘
       ▲                                   ▲
       │        요구사항/기획               │
       └────────── 4. 기획/사업 ────────────┘
                     담당
```

### 주요 협업 사항
- **1↔2**: 3D 렌더링 이미지를 결함탐지 입력으로 사용, 결함 3D 좌표 매핑
- **1↔3**: 3D 모델(.splat) 파일을 웹 뷰어로 서빙, 파이프라인 API 연동
- **2↔3**: 결함탐지/가격예측 API 설계 및 프론트엔드 표시
- **4↔전원**: 기능 우선순위 결정, 일정 조율, 사용자 피드백 반영
