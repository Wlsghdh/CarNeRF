## 프로젝트 개요

**서비스명**: CarNeRF
**슬로건**: "AI가 분석하고, 3D로 보여주는 투명한 중고차 거래"
**핵심 기술**: 3D Gaussian Splatting + AI 결함 탐지 + ChatGPT 차량 분석
**타겟 시장**: 국내 중고차 거래 시장 (2025년 기준 약 40조원 규모)

## 문제 정의 (Pain Point)

1. **정보 비대칭**: 중고차 시장에서 판매자와 구매자 간 차량 상태에 대한 정보 격차가 크다. 전문 지식이 없는 일반인은 차량 상태를 정확히 판단하기 어렵다.
2. **신뢰 부족**: 엔카, KB차차차 등 기존 플랫폼은 사진과 텍스트 기반 정보만 제공. 차량의 실제 상태를 원격으로 검증할 방법이 없다.
3. **전문 용어 장벽**: 차량 진단서, 성능기록부 등이 전문 용어로 작성되어 일반인이 이해하기 어렵다.
4. **지역 제한**: 직거래 플랫폼의 경우 같은 지역 매물을 찾기 어렵다.

## 솔루션

### 핵심 차별화: "당근마켓의 접근성 + 엔카의 전문성, 그러나 전문 지식은 AI가 대신"

1. **3D Gaussian Splatting 차량 뷰어**
   - 판매자가 스마트폰으로 차량 영상을 촬영하면 AI가 자동으로 고품질 3D 모델을 생성
   - 구매자는 웹/앱에서 360° 자유롭게 차량을 확인 가능
   - 기술: COLMAP → 3D Gaussian Splatting (A100 GPU, PSNR 31.88+)
   - 처리 시간: 일반 모드 ~45분, HQ 모드 ~140분

2. **AI 결함 자동 탐지**
   - 3D 모델에서 다각도 렌더링 → YOLOv8 기반 결함 탐지
   - 스크래치, 덴트, 도색 변색 등을 자동으로 감지
   - 3D 뷰어에서 결함 위치를 마커로 표시 (클릭 시 상세 정보)
   - 결함 심각도 점수화 → 가격 반영

3. **ChatGPT 기반 차량 AI 분석**
   - 차량 클릭 시 AI가 해당 차량의 장점/단점을 쉬운 말로 요약
   - 모델별 알려진 고질병/주의사항 자동 안내
   - 실제 구매자/판매자 후기를 종합하여 "이 차를 산 사람들의 솔직한 후기"를 AI가 요약
   - → **전문 지식 없이도** 차량 특성을 한눈에 파악 가능

4. **AI 가격 진단 + 시세 분석**
   - 동일 모델 과거 거래기록 기반 평균가격 산출
   - 감가상각 예측 (연식, 주행거리, 연료, 사고이력 반영)
   - 시세 추세 (상승/하락/보합) 시각화
   - 결함 반영 가격 자동 조정

5. **지역 매칭**
   - 사용자 등록 지역 기반으로 내 지역 매물 우선 표시
   - 지역별 필터링으로 근거리 매물 검색
   - 직거래 편의성 향상

6. **이용자 후기 시스템**
   - 구매자/판매자 모두 후기 작성 가능
   - 같은 모델의 후기를 종합하여 "이 모델의 실제 사용 후기 요약" 제공
   - → 전문가 리뷰 대신 **실제 사용자의 목소리**로 정보 제공

## 사용자 플로우

### 구매자 플로우
1. 회원가입 → 지역 설정
2. 매물 검색 (지역, 브랜드, 가격, 연료 필터)
3. 차량 클릭 → **AI 장단점 요약** 확인
4. **3D 뷰어**로 차량 360° 확인
5. **AI 파손 마커**로 결함 위치 확인
6. **시세 분석**으로 적정 가격 확인
7. **이용자 후기** 확인 (고질병 정보 포함)
8. 연락하기 → 직거래 or 안전결제

### 판매자 플로우
1. 회원가입 → **판매자 전환** (본인인증 + 차량등록증)
2. 차량 영상 촬영 (스마트폰)
3. 영상 업로드 → AI가 자동 3D 모델 생성 (~45분)
4. 매물 정보 입력 (차종, 가격, 설명)
5. AI 진단 결과 자동 첨부 → 신뢰도 향상
6. (선택) 포인트로 프리미엄 상단 배치

## 기술 스택

| 분류 | 기술 |
|------|------|
| **3D 파이프라인** | COLMAP (SfM) → 3D Gaussian Splatting (CUDA/PyTorch) |
| **AI 결함 탐지** | YOLOv8 + SAM (Segment Anything Model) |
| **AI 분석** | OpenAI ChatGPT API (gpt-4o-mini) |
| **백엔드** | FastAPI (Python 3.11) + SQLAlchemy + SQLite → PostgreSQL |
| **프론트엔드** | Jinja2 SSR + Tailwind CSS + Three.js (3D 뷰어) |
| **인프라** | NVIDIA A100 80GB x2, CUDA 12.2 |
| **향후 앱** | React Native 또는 Flutter |
